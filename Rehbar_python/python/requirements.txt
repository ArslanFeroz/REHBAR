#    gesture-controller start to %APPDATA%/RAHBAR/hand_landmarker.task
#  • numpy>=2.0 + scikit-learn>=1.5 must be installed together;
#    scikit-learn 1.4.x crashes with numpy 2.x (binary incompatibility)
